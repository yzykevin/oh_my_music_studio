Features

💻

System Information

Real-time display of platform, CPU, memory, and uptime.

🎹

DAW Detection

Auto-detect installed Digital Audio Workstations — Logic, Ableton, Pro Tools, Cubase, and more.

🔌

Plugin Scanning

Scan VST, VST3, AU, AAX plugins. Same plugin in multiple formats auto-merged into one entry.

🏭

Vendor Grouping

Plugins grouped by vendor with one-click expand/collapse. Quick search and format filter.

🔊

Hardware Panel

Audio devices, MIDI devices, Bluetooth audio, running DAWs, and login items.

📡

Driver Detection

Display audio interface driver status and configuration.

🔑

iLok Support

Detect iLok License Manager installation status.

📊

Plugin Analytics

Visualize plugin format distribution and vendor breakdown with interactive charts.

📄

Report Export

Export your full software scan as JSON or PDF for documentation.

🌙

Dark / Light Theme

One-click toggle between dark and light mode.

🌐

Bilingual UI

Built-in Chinese/English interface toggle. No configuration needed.

⚡

Parallel Scanning

Software detection runs in parallel — plugins scanned concurrently for faster startup.
